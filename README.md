<h2>📥 Execution Workspace Deployment Guide</h2>
<p>Run these sequential terminal commands inside your Termux window interface to trigger the fully automated setup engine:</p>

<!-- STEP 1 -->
<h3><img src="https://readme-typing-svg.herokuapp.com?font=Fira+Code&size=16&duration=2000&pause=500&color=00FF87&width=350&lines=%E2%9E%A4+Step+01%3A+Install+Git+Core" alt="Step 1 Animation"></h3>
<p>Updates package lists and installs the Git core package manager dependency layer.</p>
<pre><code>pkg install git -y</code></pre>

<!-- STEP 2 -->
<h3><img src="https://readme-typing-svg.herokuapp.com?font=Fira+Code&size=16&duration=2100&pause=500&color=00f2fe&width=350&lines=%E2%9E%A4+Step+02%3A+Clone+Repository" alt="Step 2 Animation"></h3>
<p>Clones the latest codebase and infrastructure configuration from the source repository.</p>
<pre><code>git clone https://github.com/ps1-blacklist/ps1-debian.git</code></pre>

<!-- STEP 3 -->
<h3><img src="https://readme-typing-svg.herokuapp.com?font=Fira+Code&size=16&duration=2200&pause=500&color=eed202&width=350&lines=%E2%9E%A4+Step+03%3A+Enter+Directory" alt="Step 3 Animation"></h3>
<p>Navigates the terminal active workspace directory straight into the project root path.</p>
<pre><code>cd ps1-debian</code></pre>

<!-- STEP 4 -->
<h3><img src="https://readme-typing-svg.herokuapp.com?font=Fira+Code&size=16&duration=2300&pause=500&color=ff416c&width=350&lines=%E2%9E%A4+Step+04%3A+Chmod+Permission" alt="Step 4 Animation"></h3>

<pre><code>chmod +x setup.sh</code></pre>

<!-- STEP 5 -->
<h3><img src="https://readme-typing-svg.herokuapp.com?font=Fira+Code&size=16&duration=2400&pause=500&color=00FF87&width=350&lines=%E2%9E%A4+Step+05%3A+Launch+Installer" alt="Step 5 Animation"></h3>

<pre><code>./setup.sh</code></pre>
